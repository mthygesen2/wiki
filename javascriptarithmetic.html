JavaScript(JS) Arithmetic

Overview

A typical thing to do with numbers is arithmetic.

Definition:

Arithmetic operators perform arithmetic on numbers (literals or variables). Such as + (addition), - (subtraction), * (multiplication), / (division), % (modulus), ++ (increment), -- (decrement).

How it Works:

Adding variable Ex. var x = 100 + 50;
Using Modulos Ex. var x = 5; var y = 2; var z = x % y;
