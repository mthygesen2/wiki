Markdown

Overview

Many people think that creating a web page involves really complicated code that requires extensive studying to understand. This is not necessarily the case. We are going to learn a simple language called Markdown in this lesson that will let you create web pages really easily. But before we do that let's quickly go over what the World Wide Web is in a nutshell.

Definition:

Markdown is a way to style text on the web. You control the display of the document; formatting words as bold or italic, adding images, and creating lists are just a few of the things we can do with Markdown. Mostly, Markdown is just regular text with a few non-alphabetic characters thrown in, like # or *.

How it Works:

Headers are constructed with # symbols and there are six sizes with one being the largest and 6 the smallest.

Ex.
# Best Chocolate Chip Cookies
## Best Chocolate Chip Cookies
### Best Chocolate Chip Cookies
#### Best Chocolate Chip Cookies
##### Best Chocolate Chip Cookies
###### Best Chocolate Chip Cookies

In Markdown there are ordered lists, which means they are numbered, and there are unordered lists, which use bullet points instead of numbers. In an unordered list, we use the * symbol followed by one space before the text. In an ordered list, we use regular numerals followed by a period and a space.

Ex.
* Butter
* Sugar
1. Beat the butter...
2. Mix the flour...
